## セットアップ

```bash
make setup    # 依存取得 + ビルド
make dev      # 開発サーバー起動
make test     # テスト
```

設定は `env/config.yaml`（非機密）、`env/secret.yaml`（ローカル秘密情報）、Doppler（チーム共有・本番秘密情報）で管理する。
`env/secret.yaml` は `.gitignore` で除外されるためコミットしない。

## ディレクトリ構成

```
.
├── env/
│   ├── config.yaml   # プロジェクト固有設定（非機密）
│   └── secret.yaml   # ローカル秘密情報（コミット禁止）
├── src/              # アプリケーションコード
└── doc/              # 仕様・設計・運用ドキュメント
```

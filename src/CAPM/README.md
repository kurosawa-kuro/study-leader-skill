# CAPM 学習ガイド

CAPM はこのリポジトリでは**取得対象**である。
役割は PM 専業化ではなく、リモート技術リードとして予算、人月、スコープ、リスク、進捗、ステークホルダー管理の会話ができることを外部証明する点にある。

2. [`02_スコープ_スケジュール_コスト.md`](./02_スコープ_スケジュール_コスト.md)
3. [`03_リスク_品質_変更管理.md`](./03_リスク_品質_変更管理.md)

## 学習方針

- 23 時間教育要件を満たす
- 合格点突破型で進める
- 用語暗記だけで終わらせず、実務会話に戻せる形で理解する
- 技術主軸を壊さない範囲で進める

## 到達目標

- CAPM の受験要件と試験形式を説明できる
- スコープ、スケジュール、コスト、リスク、品質、変更管理の会話ができる
- 「技術リードとしてなぜ CAPM を取るのか」を一言で説明できる

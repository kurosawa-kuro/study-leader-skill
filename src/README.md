# src/ — 学習コンテンツ

このディレクトリは、リーダースキル資格戦略の実体となる学習コンテンツを置く。
`doc/` が方針と設計の正本であり、`src/` はその方針に従って読む教材本体を置く場所である。

## 構成

- `PSPO-I/` — 学習のみで回収する語彙群
- `CAPM/` — 取得前提で進める PM 管理観点の教材

## 読み順

1. `PSPO-I/README.md`
2. `PSPO-I/01_価値判断と優先順位.md`
3. `PSPO-I/02_Product_BacklogとGoal.md`
4. `PSPO-I/03_ステークホルダー調整と学習のみ判断.md`
5. `CAPM/README.md`
6. `CAPM/01_受験要件と学習戦略.md`
7. `CAPM/02_スコープ_スケジュール_コスト.md`
8. `CAPM/03_リスク_品質_変更管理.md`
9. `CAPM/04_ステークホルダー_コミュニケーション.md`

## ルール

- Markdown を原本とする
- `doc/01_仕様と設計.md` の 8 項目テンプレートを崩しすぎない
- `PSPO-I` は受験対策本ではなく語彙回収用の教材として扱う
- `CAPM` は合格点突破と実務説明力の両立を目的とする

# PSPO-I 学習ガイド

PSPO I はこのリポジトリでは**受験対象ではなく学習対象**である。
目的は Product Owner 観点の語彙を回収し、CAPM 前に「価値判断の言葉」を手に入れることにある。

## 位置づけ

- 価値判断、優先順位、Backlog 整理の語彙を得る
- 現場評価で代替できるため、今回は外部証明としては使わない
- 学習時間は 5〜10h を目安にし、CAPM や技術学習を圧迫しない

## 読み順

1. [`01_価値判断と優先順位.md`](./01_価値判断と優先順位.md)
2. [`02_Product_BacklogとGoal.md`](./02_Product_BacklogとGoal.md)
3. [`03_ステークホルダー調整と学習のみ判断.md`](./03_ステークホルダー調整と学習のみ判断.md)

## ゴール

- 顧客価値、優先順位、Backlog、Goal、ステークホルダー調整を自分の案件で言い換えられる
- 「なぜ PSPO I は受験ではなく学習のみでよいか」を説明できる
